SluggedBehavior::removeStopWords

removeStopWords from a string. if $splitOnStopWord is true, the following occurs: input "apples bananas pears and red cars" output array('apples bananas pears', 'red cars')

If the passed string doesn't contain the separator, or after stripping out stop words there's nothing left - the original input is returned (in the desired format) Therefore passing "contain" will return immediately array('contain') Passing "contain this text" will return array('text') both contain and this are stop words Passing "contain this" will return array('contain this')
