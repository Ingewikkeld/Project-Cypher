SluggedBehavior::slug

slug method

For the given string, generate a slug. The replacements used are based on the mode setting, If tidy is false (only possible if directly called - primarily for tracing and testing) separators will not be cleaned up and so slugs like "-----as---df-----" are possible, which by default would otherwise be returned as "as-df". If the mode is "id" and the first charcter of the regex-ed slug is numeric, it will be prefixed with an x.
