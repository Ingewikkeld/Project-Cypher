RandomLib::random

Generates a random string of a given type and length.

$str = Text::random(); // 8 character random string The following types are supported: alnum : Upper and lower case a-z, 0-9 alpha : Upper and lower case a-z hexdec : Hexadecimal characters a-f, 0-9 distinct : Uppercase characters and numbers that cannot be confused You can also create a custom type by providing the "pool" of characters as the type.
