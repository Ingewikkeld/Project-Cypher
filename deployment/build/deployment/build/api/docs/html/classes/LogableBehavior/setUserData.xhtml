LogableBehavior::setUserData

Use this to supply a model with the data of the logged in User.

Intended to be called in AppController::beforeFilter like this : if ($this->{$this->modelClass}->Behaviors->attached('Logable')) { $this->{$this->modelClass}->setUserData($activeUser);/ } The $userData array is expected to look like the result of a User::find(array('id'=>123));
