LogableBehavior::findLog

Useful for getting logs for a model, takes params to narrow find.

This method can actually also be used to find logs for all models or even another model. Using no params will return all activities for the models it is called from. Possible params : 'model' : mixed (null) String with className, null to get current or false to get everything 'action' : string (null) String with action (add/edit/delete), null gets all 'order' : string ('created DESC') String with custom order 'conditions : array (array()) Add custom conditions 'foreign_id' : int (null) Add a int (remember to use your own user key if you're not using 'user_id') 'user_id' : int (null) Defaults to all users, supply id if you want for only one User
