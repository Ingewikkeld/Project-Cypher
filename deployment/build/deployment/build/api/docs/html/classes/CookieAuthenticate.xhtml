\CookieAuthenticate

An authentication adapter for AuthComponent. Provides the ability to authenticate using COOKIE

{{{
$this->Auth->authenticate = array(
'Authenticate.Cookie' => array(
'fields' => array(
'username' => 'username',
'password' => 'password'
),
'userModel' => 'User',
'scope' => array('User.active' => 1),
'crypt' => 'rijndael'// Defaults to rijndael(safest), optionally set to 'cipher' if required
)
)
}}}

    Methods

    public

    • authenticate() — Authenticates the identity contained in the cookie. Will use the `settings.userModel`, and `settings.fields` to find COOKIE data that is used to find a matching record in the `settings.userModel`. Will return false if there is no cookie data, either username or password is missing, of if the scope conditions have not been met.
