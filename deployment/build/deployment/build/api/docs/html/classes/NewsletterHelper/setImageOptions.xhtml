NewsletterHelper::setImageOptions

Get the right dimension for the orientation of the images in the newsletter.

The width and height get set whether the image is of landscape or portrait oriented. These values will then be added to $this->imgOptions in both CSS style as inline 'height' and 'width' attributes for the <img> tag
